# SkedCheck

**Airline Pilot Schedule Viewer & FAR 117 Checker**

This tool is designed to help pilots visualize their schedules and check compliance with FAR 117 block and FDP limits. It compares both past and scheduled flights, showing the lowest available limitation times for each day.
